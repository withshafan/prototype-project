# Green Grocer – Premium Organic Prototype

A polished landing-page prototype for an organic grocery brand built as a single-page HTML/CSS experience.

## Overview

- **Project:** Premium organic grocery mobile UI concept
- **Built with:** HTML and CSS only
- **File:** `index.html`

## Features

- Mobile phone frame with dynamic island and screen transitions
- Premium green/gold organic palette
- Minimal, modern product showcase design
- Responsive-friendly layout for mobile preview

## How to view

1. Open `index.html` in your browser
2. Or use a local web server if desired

## Notes

This repository currently contains a single static prototype page. No build tools or dependencies are required.
